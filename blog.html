Alex Clapperton

21/10/2015

The Layout

I had decided from the very start of this project that I wanted to move in the direction of using a full width one column layout. The reason behind this decision is for the captivation that the site will receive from visitors. It also gives more space to the content that is present on each page, as using a multiple column layout can make the content look squashed and make the entire site look cramped.

By using a full width one column layout, it linked up respectively with Parallax scrolling. Parallax scrolling involves the background moving at a slower rate to the foreground, creating a 3D effect as you scroll down the page. It can sometimes be overwhelming, but when used sparingly it can provide a nice, subtle element of depth. I was able to acquire a Parallax JavaScript plugin on GitHub from PixelCog Inc. that I implemented at the top section of each page on the site.

The Design

For the design I decided to implement my favourite colour scheme of red, dark grey, white & light grey. I believe these colours blend very well together and create a strong contrast between them, giving the website a professional, clean and sophisticated look.

Hex Codes: #d40000, #222222, #000000, #2c2c2c.

For my logo, I decided to use Adobe Illustrator as you can easily work with vectors. Using vectors means that no matter how much you scale up or down the content you create in Illustrator, the content will stay crisp and sharp which is a real benefit when creating logos. My design of the logo is very simple, consisting of only my initials A M C, which I believe matches the look and feel of my website. I had to create two versions of my logo, one black version and one white version, beacuse as the website scrolls, the navigation bar's position becomes fixed and the background of the navigation bar changes from transparent to white. As the navigation bar becomes white I had to write some JavaScript that changed the logo from white to black.

JavaScript code adds class 'dark' to navigation as user scrolls.

CSS used to change logo from white to black.

HTML & CSS Code

For the HTML coding of the website I wanted to make sure the code was kept as clean as possible. To achieve this I used large comments to highlight the most important parts of the website, such as commenting when a section of a page starts and ends, when the navigation starts and ends etc. I also used smaller comments to highlight the start and end of each HTML tag element. By doing this I feel it is easier to navigate through the code and locate the correct opening and closing tag elements.

Large and small comments of an opening and closing HTML tag element.

In terms of the CSS coding, I decided to follow the CSS guidlines of the well known Harry Roberts. By using his methods I feel my code is very clean and can be easily read & understood. As well as organising my CSS code into sections, I have implented a table of contents comment section at the top of the stylesheet so it is easy to navigate through the code.

CSS coding and commenting using Harry Roberts methods.

Summary

Overall I feel my website matches the correct standards and has a clean and professional look. It is HTML validated, the coding is clean and readable and the coding is very well commented.

Nearly all the images are lower than 25KB in size except for the main banner images at the top of each page, as I had to compensate a bit of extra image size to get a good enough quality image.
